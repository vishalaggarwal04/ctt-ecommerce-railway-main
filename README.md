# # AmCart E-Commerce Application

React e-commerce app where you can **browse products WITHOUT login**, just like Amazon!

## 🎯 Perfect Flow

```
Open App → Shop Page
   ↓ Browse, search, filter (NO LOGIN!)
   ↓ Click "Add to Cart"
   ↓ Prompt to login
   ↓ Login → Add products → Checkout
```

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Opens at: **http://localhost:3000/shop**

## ✨ Key Features

✅ **Browse WITHOUT Login** - See all products
✅ **Login When Needed** - Only for cart/checkout  
✅ **Advanced Search** - Filter, sort, search
✅ **Guest Banner** - "Login to add to cart"
